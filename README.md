# GFNFF Parameter Optimisation

Tools for fine‑tuning GFNFF parameters using GULP and soap descriptors.
This project makes use of GULP version 6.4 with its pGFNFF module (Periodic Generic Force Field)
## Structure
- `src/` – main Python package
- `scripts/` – utility scripts
- `tests/` – benchmarks and experiments
- `data/` – input CIF files
- `results/` – output databases (ignored by git)
